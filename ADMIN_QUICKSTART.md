# 🚀 Admin Quick Start - 3 Phút Tạo Trang Mới

## 1️⃣ Mở Admin (10 giây)

Mở file trong trình duyệt:
```
d:\elgacho\admin\index.html
```

Hoặc deploy lên web:
```
https://vck2910.id.vn/admin/
```

## 2️⃣ Kéo Thả Components (2 phút)

### Quick Recipe cho Trang Restaurant:

**Header:**
```
Kéo: Navbar
```

**Hero Section:**
```
Kéo: Section
  → Thay text: "Welcome to El Gaucho"
  → Style: Background #C41E3A, Text white, Padding 100px
```

**About Section:**
```
Kéo: Section
  → Kéo: Container
    → Kéo: Row (2 columns)
      → Column 1: Image
      → Column 2: Text
```

**Menu Section:**
```
Kéo: Section
  → Kéo: Menu Tabs
  → Kéo: Gallery (cho món ăn)
```

**Contact Section:**
```
Kéo: Section
  → Kéo: Form
  → Kéo: Map
  → Kéo: Social Links
```

## 3️⃣ Export & Deploy (30 giây)

**Export:**
```
Click "Export HTML" → Download file
```

**Deploy lên GitHub:**
```powershell
# Đổi tên file vừa download thành index.html
# Copy vào d:\elgacho\

# Trong PowerShell:
cd d:\elgacho
git add .
git commit -m "Update page from admin builder"
git push origin main
```

**Chờ 2-3 phút** → Website cập nhật!

## ⚡ Phím Tắt Quan Trọng

| Phím | Tác Dụng |
|------|----------|
| `Ctrl + Z` | Hoàn tác |
| `Ctrl + S` | Lưu |
| `Ctrl + D` | Sao chép |
| `Delete` | Xóa |

## 💡 Pro Tips

### Tip 1: Sao Chép Nhanh
```
Tạo 1 card đẹp → Ctrl+D → Sao chép 10 cái
Chỉ cần đổi nội dung!
```

### Tip 2: Test Responsive
```
Xây trên Desktop → Click Mobile icon → Xem trên điện thoại
```

### Tip 3: Undo Unlimited
```
Làm sai gì cũng có thể Ctrl+Z!
```

## 📱 Responsive Auto

Admin tự động tạo responsive:
- Desktop: Full width
- Tablet: Tự động scale
- Mobile: Stack columns

Không cần code thêm!

## 🎨 Color Palette Restaurant

Copy paste các màu này vào Style tab:

**Primary:**
- `#C41E3A` - Đỏ (như El Gaucho)

**Neutral:**
- `#333333` - Text đen
- `#666666` - Text xám
- `#F5F5F5` - Background sáng
- `#FFFFFF` - Trắng

**Accent:**
- `#D4AF37` - Vàng gold
- `#006400` - Xanh lá
