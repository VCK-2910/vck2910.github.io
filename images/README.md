# El Gaucho Restaurant Website

Đây là placeholder cho thư mục images. Vui lòng thêm các hình ảnh của nhà hàng vào đây.

## Hình ảnh cần thiết:

### 1. Hero Background
- **Tên file:** `hero-bg.jpg`
- **Kích thước đề xuất:** 1920x1080px
- **Mô tả:** Hình ảnh nền cho phần hero (banner chính)

### 2. Không gian nhà hàng
- **Tên file:** `restaurant-interior.jpg`
- **Kích thước đề xuất:** 800x600px
- **Mô tả:** Hình ảnh không gian bên trong nhà hàng

### 3. Món ăn (Gallery)
- `dish1.jpg` - Hình ảnh món bít tết hoặc món chính
- `dish2.jpg` - Hình ảnh món hải sản
- `dish3.jpg` - Hình ảnh món tráng miệng

### 4. Không gian (Gallery)
- `interior1.jpg` - Khu vực ăn uống chính
- `interior2.jpg` - Khu vực VIP
- `bar.jpg` - Quầy bar

## Lưu ý:
- Tất cả hình ảnh nên có chất lượng cao
- Format: JPG hoặc PNG
- Tối ưu kích thước file để website tải nhanh (nên dưới 500KB mỗi ảnh)
- Sử dụng công cụ nén ảnh như TinyPNG hoặc ImageOptim

## Nguồn hình ảnh miễn phí (nếu cần):
- Unsplash.com
- Pexels.com
- Pixabay.com

Tìm kiếm với từ khóa: "restaurant", "steak", "food", "dining", "interior"
